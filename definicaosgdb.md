Dados: informações ou fatos que podem ser armazenados e processados por um sistema de computador. Os dados podem estar em diferentes formatos, como texto, números, imagens, áudio, entre outros.

Banco de dados: é um conjunto organizado de dados que são armazenados e gerenciados em um sistema de computador. Um banco de dados pode ser usado para armazenar informações sobre pessoas, produtos, transações, eventos, entre outros, e pode ser acessado e atualizado por meio de um Sistema de Gerenciamento de Banco de Dados (SGBD).

Modelo de dados: é um conjunto de conceitos e regras que define a estrutura e as relações entre os dados em um banco de dados. Existem vários modelos de dados, como o modelo hierárquico, o modelo em rede, o modelo relacional, o modelo orientado a objetos, entre outros.

Metadados: são dados que descrevem outros dados. Eles fornecem informações sobre o conteúdo, estrutura e contexto dos dados armazenados em um banco de dados. Os metadados podem incluir informações como nome da tabela, tipo de dado, tamanho, índices, restrições, entre outros.

Catálogo de dados: é um conjunto de metadados que descreve a estrutura de um banco de dados e seus objetos, como tabelas, índices, visões e procedimentos armazenados. O catálogo de dados é usado pelo SGBD para gerenciar o banco de dados e permitir que os usuários acessem e atualizem os dados armazenados.
